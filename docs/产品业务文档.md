# 梗百科 - AI热词解码器

> **TRAE AI 创造力大赛参赛作品**  
> 赛道：生活娱乐 | 作者：单人 | 开发工具：TREA IDE

---

## 一、产品简介

**梗百科**是一款 AI 驱动的热词解释工具，专为解决"信息过载时代跟不上流行语"的痛点而生。用户输入任何看不懂的网络热梗、专业术语、新兴词汇，AI 即刻给出通俗易懂的解释，包含含义、出处、使用场景和相关推荐。

核心理念：**不懂任何词，来这就对了。**

---

## 二、解决什么问题

### 痛点

- 互联网每天涌现大量新词热梗（如"内卷""破防""显眼包"等），普通人难以跟上
- 搜索引擎查找效率低，结果碎片化
- 专业术语壁垒高，外行人看不懂百科式的定义
- 没有一款专门针对"网络热词即时解释"的工具

### 解决方案

使用 AI 大模型，以对话式、结构化的方式解释任何词汇，让用户 10 秒内就能完全理解一个新词。

## 三、核心功能

| 功能模块 | 说明 |
|----------|------|
| AI 智能解释 | 输入词/短语，AI 返回含义、出处、例句、分类标签 |
| 语音输入 | 支持中文语音识别，懒得打字也能搜 |
| 热词标签云 | 首页展示热门搜索词，一键点击查看 |
| 热词排行榜 | 按真实搜索热度动态排序的 TOP 15 榜单 |
| 自生长知识库 | AI 每解释一个新词，自动沉淀入库，搜索越多越聪明 |
| 收藏夹 | 收藏感兴趣的解释，随时回顾 |
| 分享卡片 | 一键生成精美解释卡片，可复制分享 |
| 暗色/亮色主题 | 护眼模式自由切换 |
| PWA 安装 | 一键安装到桌面，像原生软件一样使用 |
| 多 API 接入 | 支持 DeepSeek/OpenAI/自定义接口，前端直接配置 |

## 五、技术架构

```
┌─────────────────────────────────────────────┐
│                   前端 (SPA + PWA)            │
│  index.html + style.css + app.js             │
├─────────────────────────────────────────────┤
│  AI Engine                                   │
│  ┌─────────┬──────────┬─────────────────┐   │
│  │ 知识库  │ LLM API │ 联网搜索(DDG)    │   │
│  │ (种子+  │ (Deep-  │ (兜底)           │   │
```

- **前端**：原生 HTML/CSS/JS 单页应用，零框架依赖
- **AI 引擎**：三级策略（知识库 → API → 联网搜索 → 模板兜底）
- **PWA**：Service Worker + manifest.json，支持离线使用和桌面安装
- **存储**：纯前端 localStorage，无需后端数据库

## 六、创新亮点

### 1. 自生长知识库
不是静态词库，而是"活的"系统。每解释一个新词，自动入库；搜索次数驱动热词排行；用得越多越智能。

### 2. 三级智能降级策略
```
LLM API → 联网搜索 → 模板生成
  (最优)     (备用)      (保底)
```
无论有无 API Key、有无网络，都能正常工作。

### 3. 前端直配 API
不需要改代码，在页面上直接粘贴 API Key 即可切换 AI 模型。降低使用门槛，也符合比赛"展示给评委"的场景。

### 4. PWA 双形态
既是网页也是桌面软件。扫码打开即用，一键安装变成本地应用。
